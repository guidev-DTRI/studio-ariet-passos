# Limpeza de pele | Ariane Tadday

Sistema web desenvolvido para auxiliar a organização de atendimentos, agendamentos e clientes de Limpeza de pele | Ariane Tadday.

## Objetivo

Criar uma aplicação simples, elegante e funcional para melhorar o fluxo de atendimento, organizar horários e facilitar a comunicação com clientes via WhatsApp.

## Funcionalidades previstas

- Página pública do studio
- Painel administrativo
- Cadastro básico de clientes
- Cadastro de procedimentos
- Agenda de horários
- Confirmação via WhatsApp
- Histórico simples de atendimentos

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
