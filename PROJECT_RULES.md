# Regras do Projeto — Studio Ariet Passos

## Objetivo
Criar um sistema web elegante para gerenciamento de atendimentos do Studio Ariet Passos.

## Tecnologias atuais
- HTML
- CSS
- JavaScript

## Futuro do projeto
O projeto poderá futuramente:
- utilizar frameworks modernos;
- integrar banco de dados;
- utilizar autenticação;
- utilizar React;
- evoluir para PWA ou aplicação SaaS.

## Regras gerais
- Priorizar código organizado
- Priorizar segurança e privacidade
- Não coletar dados sensíveis inicialmente
- Manter visual premium feminino
- Interface clean e elegante
- Código escalável e reutilizável

## Identidade visual
- Bege
- Nude
- Branco
- Dourado suave
- Marrom elegante
- Estética minimalista premium
