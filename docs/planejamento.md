# Planejamento — Studio Ariet Passos

## Objetivo do projeto

Criar um sistema web para organizar os atendimentos do Studio Ariet Passos, começando pelo painel administrativo.

## Público-alvo

Clientes interessadas em procedimentos de estética facial e sobrancelhas.

## Funcionalidades iniciais

### Painel administrativo

- Ver agenda do dia
- Cadastrar cliente
- Cadastrar procedimento
- Criar horários disponíveis
- Confirmar atendimento pelo WhatsApp

### Página pública

- Apresentar o studio
- Mostrar serviços
- Botão para agendamento
- Link para Instagram e WhatsApp

## Dados coletados inicialmente

- Nome
- WhatsApp
- Procedimento
- Data
- Horário
- Status do atendimento

## Dados que NÃO serão coletados nesta fase

- CPF
- Endereço
- Fotos de rosto
- Informações médicas
- Alergias
- Medicamentos
- Ficha de anamnese completa
